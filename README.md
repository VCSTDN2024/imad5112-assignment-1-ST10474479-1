The apps primary purpose is to help the user find quick and easy meal suggestions especially if one has a busy hectice schedule

3. App Purpose and Features

## Purpose and Features

The main goal of this app is to show case meals based on the time of day the user inputs. This makes things easier as the user does not have to think of what meal to make

### Key Features:

- Feature 1: User friendly; User is able to input the time of day based on what time they would like to eat
- Feature 2: There is a reset button to clear the meals displayed and a user is able to input another time of day
- Feature 3: Exit button to exit the entire application once user is satisfied

  These feature aim to provide a simple user friendly interface and very easy going platform for managing tasks
